Projects

Book Corner (Final Year Project)

University of the Punjab, Pakistan

The Book Corner is an android based application about the various book stores and publishers to make their books available online for android users developed as a final year project of our BS studies. The application is developed for book readers so that they can view the different books, browse for different books, and also get the information about incoming and available stock. The users can also place the order and add the books to wish-list for further purchasing. The application is user-friendly, with graphical screens and have support to directly chat with the vendor.

The developed project also deals with the e-Commerce area. The developed project contains several modules, we have used different technologies for developing the application, and different SDKs have been used to implement several APIs. The project is state of the art application and can be further enhanced by implementing other modules.
